# AI and Text Analytics Labs

MSc Data Science coursework and lab exercises for the Introduction to AI and Text Analytics unit.

## Repository Structure

### 01_introduction
- Introduction to AI concepts
- Performance metrics and evaluation

### 02_supervised
- **k-NN**: k-Nearest Neighbors implementation
- **Naive Bayes**: Classification algorithms  
- **Regression**: Linear regression with gradient descent
- **Logistic Regression**: Binary classification
- **Classifiers**: General classification techniques

### 03_PCA
- Principal Component Analysis
- Dimensionality reduction techniques
- Irish election data analysis examples

### 04_unsupervised
- **k-means**: Clustering algorithms  
- **Picking k**: Methods for choosing optimal cluster count
- **HAC**: Hierarchical Agglomerative Clustering

### 05_regression
- Linear regression
- Logistic regression
- Classification methods

### 06_neural_networks
- Perceptron
- Multi-layer perceptron (MLP)
- Training methods

### worksheets
Practical exercises and solutions covering all topics above.

## Data Sources

- Irish election data (2020 general election)
- Hawks morphological data
- Programming language similarity data  
- Simulated datasets for various ML techniques

## Technologies

- Python (NumPy, Pandas, Matplotlib, Scikit-learn)
- LaTeX for documentation
- Jupyter Notebooks for interactive examples
